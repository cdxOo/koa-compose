
# @cdxoo/koa-compose

**IMPORTANT: this is a fork of koajs/compose so i can do watever with it DONT USE IT**

 Compose middleware specifically for Koa.

## Installation

```js
$ npm install @cdxoo/koa-compose
```

## API

### fn = compose([a, b, c, ...])

  Compose the given middleware and return middleware.
